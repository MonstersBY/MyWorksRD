JANUS Pro Sunlike (Южная Корея)

Janus Pro Sunlike – диагностический аппарат последнего поколения, которой визуализирует несовершенства кожи и определяет зоны, требующие особого ухода.

Аппарат позволяет проводить диагностику кожи по 18 параметрам с помощью усовершенствованной системы Sunlike LED и UV LED 385 нм.

Применение
