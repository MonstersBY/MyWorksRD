Описание

Cварочные полуавтоматы на базе микропроцессора Ehave CM (CO2/MAG/MMA) предназначены для сварки металлической проволокой и ручной дуговой сварки плавящимся электродом. Аппараты широко применяются в различных отраслях при сварке углеродистой стали: железнодорожный транспорт, судостроение, автомобилестроение, производство стальных конструкций, контейнеров, металлообработке, металлической мебели.

Основные преимущества:
